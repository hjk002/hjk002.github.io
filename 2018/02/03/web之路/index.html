flex 布局

采用flex布局，它的所有子元素自动成为容器成员，任何一个项目都可以指定为为flex布局。

容器的属性

###flex-direction

flex-direction 排列盒子方向

row:横向排列（从左到右） 默认
row-reverse: 横向排列 从右往左排 与row相反

flex-direction 排列盒子方向
row(默认)： 横向从左到右排列（横向 ）。
row-reverse： 对齐方式与row相反。 （倒序排列）
column： 纵向从上往下排列（纵向）。
column-reverse：对齐方式与column相反。 （倒序排列）
